Sobre o Visualizador do PowerPoint

O Visualizador do Microsoft Office PowerPoint permite exibir apresentações completas criadas no Microsoft Office PowerPoint 2003. Ele é compatível com apresentações criadas no PowerPoint 97, PowerPoint 2000 e PowerPoint 2002. Além disso, ele permite abrir apresentações do PowerPoint protegidas por senha.

Requisitos do sistema

  • PC com processador de 200 MHz ou superior
  • Microsoft Internet Explorer 5 ou posterior
  • Sistemas operacionais compatíveis: Microsoft Windows 98 Segunda Edição, Windows Millennium Edition, Windows 2000 com Service Pack 3, Windows XP ou Windows Server 2003.
  • Memória RAM:
    • Para Windows 98 SE e Windows ME: 32 MB de RAM
    • Para Windows 2000 com Service Pack 3: 64 MB de RAM
    • Para Windows XP e Windows Server 2003: 128 MB de RAM
  • 5 MB de espaço disponível no disco rígido
  • Monitor com resolução Super VGA (800 x 600) ou superior

Questões multilíngües

O Visualizador do PowerPoint não oferece suporte ao produto Revisores de Texto do Microsoft Office.

Ele não oferece suporte a apresentações em idioma bidirecional criadas no PowerPoint 97.

Restrições do Visualizador do PowerPoint

Não mostra as animações de diagrama corretamente. Em vez de mostrar as animações, o diagrama aparece concluído.

Não aceita abrir apresentações do Gerenciamento de Direitos de Informação (IRM).

Não oferece suporte à execução de macros e programas nem aceita abrir objetos vinculados ou incorporados.

Não oferece suporte à exibição dos seguintes formatos gráficos quando vinculados em uma apresentação: .eps, .pct, .emz, .cdr, .cgm, .wpg e .wmz.
